# Ledger â€” Local AI Resource Governor Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A standalone read-only terminal dashboard that reads Claude Code session transcripts off disk and shows live token usage, dollar cost, and system resource consumption (CPU/RAM/GPU) for all Claude sessions on the machine, with dollar budget alerts.

**Architecture:** Pure-Python package `ledger/` (sibling to `dispatch_hub`, no imports of it). A parse layer turns Claude Code JSONL transcripts into typed `UsageEvent`s; a pricing module converts them to dollars; an aggregator folds them into rollups; a watcher tails files for live updates; a resources module snapshots GPU/CPU/RAM; a `rich` TUI renders it all on a ~1.5s tick. GPU comes from the `nvidia-smi` subprocess (no binding dependency).

**Tech Stack:** Python 3.14 (`C:/Python314/python`), `rich` (TUI), `psutil` (per-process), `nvidia-smi` (subprocess), `pytest`.
